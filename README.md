# Sistema de Gestão da Copa do Mundo 2026

Projeto desenvolvido para a disciplina de Técnicas de Programação 1 (CIC0197) da Universidade de Brasília (UnB).

## Descrição

Sistema computacional para apoiar e automatizar os principais processos relacionados à organização e gestão da Copa do Mundo de 2026, abrangendo o gerenciamento de seleções, jogadores, partidas, estádios, arbitragem e controle de ingressos.

## Tecnologias

- Java
- Maven
- IntelliJ IDEA
- Swing

## Estrutura do Projeto
src/main/java/br/unb/cic0197/copa2026/
├── app/          # Ponto de entrada da aplicação
├── controller/   # Ponte entre a view e o service
├── model/        # Entidades do domínio
├── view/         # Telas da aplicação
├── service/      # Regras de negócio
├── repository/   # Persistência de dados em arquivos
└── exception/    # Exceções personalizadas

## Como Executar

1. Clone o repositório
2. Abra no IntelliJ IDEA
3. Execute a classe `Main`
